CDCS Cross-Device Test

v2 — Browser SDK 2.12.1

Tests cross-device consent sharing between two different Settings IDs (simulating mobile → web).
Device A: init + accept all → capture controllerID.
Device B: init with that controllerID on a different Settings ID → verify consent is restored and shouldCollectConsent is false.
Respects the one-time restoration rule: each test pair uses a fresh controllerID.
